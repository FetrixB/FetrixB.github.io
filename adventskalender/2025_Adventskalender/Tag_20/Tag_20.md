# Tag 20

## Story
Die Meister-Handwerkerin der Elfen steht vor einer kaputten Crafting-Station! "Unsere magischen Crafting-Rezepte sind alle verschwunden!", ruft sie verzweifelt. "Schere + Papier sollte Geschenkpapier ergeben, Kerze + Halter sollte ein Windlicht werden, aber nichts funktioniert mehr!" Du erkennst das Problem sofort: Die Crafting-Station braucht eine dynamische Benutzeroberfläche, die Items per Drag & Drop kombiniert und automatisch neue Elemente generiert. Mit createElement und intelligenten Attribut-Systemen kannst du eine Crafting-Table-UI erschaffen, die genauso intuitiv funktioniert wie die Original-Workbench aus Minecraft!

## Lernen

### createElement für dynamische Interface-Generierung
- HTML-Elemente programmatisch erstellen und konfigurieren
- Attribute mit setAttribute gezielt setzen und ändern
- CSS-Klassen dynamisch zuweisen für sofortiges Styling

### Attribute-Management für Daten-Übertragung
- Data-Attributes für custom Element-Informationen
- GetAttribute und hasAttribute für Zustandsabfragen
- RemoveAttribute für saubere Element-Bereinigung

### Komplexe UI-Systeme mit Event-Koordination
- Drag & Drop API für natürliche Bedienung
- State-Management für Crafting-Inventar verwalten
- Validation-Logic für gültige Rezept-Kombinationen

## Aufgabe
Dynamic Crafting Table UI bauen.

## Ziel-WOW
Minecraft-ähnliches Crafting-Interface.

## Libraries
TailwindCSS, Vanilla JS
